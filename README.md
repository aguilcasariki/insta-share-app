# Insta Share App

## Installation

Use the package manager [pnpm](https://pnpm.io/) to install Insta Share App.

```bash
pnpm install
pnpm run dev
```

## Usage

Open [http://localhost:3000](http://localhost:3000) with your browser to see the App.
